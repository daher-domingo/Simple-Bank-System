# Simple Bank System

A web-based banking system built with PHP and MySQL that allows users to manage accounts, perform deposits and withdrawals, and view transaction records. The system features a clean, responsive, and user-friendly interface for smooth banking operations.

---

## 🌐 Live Demo
👉 https://simplebanksystem.great-site.net/index.php

---

## 📌 Features
- User account management  
- Deposit and withdrawal transactions  
- Transaction history tracking  
- Simple and clean UI  

---

## 🛠️ Tech Stack
- HTML, CSS, Bootstrap  
- JavaScript, jQuery (AJAX)  
- PHP  
- MySQL

## 📁 Project Setup (Localhost)

If you want to run this project locally:

1. Install XAMPP or WAMP  
2. Copy the project folder to:  
   `htdocs/simple-bank-system`  
3. Import the database file into phpMyAdmin  
4. Start Apache and MySQL  
5. Open:  
   `http://localhost/simple-bank-system`
